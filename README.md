# weibonlp
#### 功能介绍
weiboAPI.py功能包含：
- 通过调用微博API的方法将微博评论写入数据库

xueweibo.py功能包含：
- 爬取微博评论并写入数据库

xuenlp.py功能包含：
- 读取数据库并进行数据去重
- 对微博评论进行情感分析并生成统计结果
- 统计微博评论中的表情排行
- 统计微博评论中的粉丝排行前20
